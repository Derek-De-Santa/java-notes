Java Basics

Java is an object-oriented, high-level, and platform-independent programming language.

  • Originally named Oak by James Gosling
  • Now maintained by Oracle Corporation

Installing Java

  1. Download and install JDK (Java Development Kit)
  2. Install an IDE like IntelliJ IDEA, Eclipse, or VS Code
  3. Set up environment variables
  4. Write and run Java programs

Hello World Program

class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Compiling and Running Java

  • javac MyProgram.java - Compiles the code
  • java MyProgram - Runs the program
